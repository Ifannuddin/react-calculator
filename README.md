# React Calculator

A simple calculator app built with React and Vite.

## Prerequisites / Prasyarat

Before running this project, make sure you have Node.js installed on your system.  
Sebelum menjalankan proyek ini, pastikan Node.js sudah terinstal di sistem Anda.

- Node.js (version 14 or higher / versi 14 atau lebih tinggi)
- npm (comes with Node.js / sudah termasuk dengan Node.js)

## Installation / Instalasi

1. Clone this repository / Clone repositori ini:
```bash
git clone https://github.com/Ifannuddin/react-calculator.git
cd react-calculator
```

2. Install dependencies / Instal dependensi:
```bash
npm install
```

## How to Run / Cara Menjalankan

### Development Mode / Mode Pengembangan

To run the website in development mode with hot reload:  
Untuk menjalankan website dalam mode pengembangan dengan hot reload:

```bash
npm run dev
```

The website will be available at `http://localhost:5173` (or another port if 5173 is busy).  
Website akan tersedia di `http://localhost:5173` (atau port lain jika 5173 sedang digunakan).

### Build for Production / Build untuk Produksi

To create a production build:  
Untuk membuat build produksi:

```bash
npm run build
```

The built files will be in the `dist` folder.  
File hasil build akan berada di folder `dist`.

### Preview Production Build / Preview Build Produksi

To preview the production build locally:  
Untuk melihat preview build produksi secara lokal:

```bash
npm run preview
```

## Available Scripts / Script yang Tersedia

- `npm run dev` - Start development server / Jalankan server pengembangan
- `npm run build` - Build for production / Build untuk produksi
- `npm run preview` - Preview production build / Preview build produksi

## Technology Stack / Teknologi yang Digunakan

- React 18
- Vite 5
- CSS
